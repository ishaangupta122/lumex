# Fluxion

A modern design collaboration platform landing page with smooth animations and interactive components.

## Tech Stack

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **GSAP** - Advanced animations
- **Lenis** - Smooth scroll

## Getting Started

```bash
npm install
npm run dev
```
